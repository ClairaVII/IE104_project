## Chạy project
Thực hiện các lệnh sau theo thứ tự:

1. Clone repository:
```bash
git clone https://github.com/hoangphuc2611/IE104_project
```

2. Di chuyển vào thư mục dự án:
```bash
cd IE104_project
```

3. Cài đặt dependencies (Yêu cầu có Nodejs) :
```bash
npm install
```

4. Chạy cục bộ:
```bash
npm start
```

4. Chạy cục bộ:
```bash
npm start
```
